

# Introduction

这个项目用于收集平时工作中使用的比较多，比较通用的一些工具类及算法，尽量将其精炼，以形成我自己的代码库。

# Prerequisite

* maven3.2.3
* jdk1.7

# Content 
+ com-gansuer-algorithm : 一些常用的数据结构及算法实现  
+ com-gansuer-file : 一些常用的文件操作的工具类
+ com-gansuer-http : 整理出来一个处理Http请求的工具类
+ com-gansuer-string : 字符串操作常用的类以及算法
